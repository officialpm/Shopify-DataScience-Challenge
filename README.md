# Shopify Data Science Intern Challenge (Winter 2019)

A small, self-contained analysis notebook for the **Shopify Winter 2019 Data Science Intern Challenge**.

This repo contains the original dataset (CSV) and a Jupyter notebook with exploratory analysis + answers.

## What’s inside

- `Shopify_Data_Science_Challenge__Parth_Dilip_Maniar.ipynb` — main notebook
- `2019 Winter Data Science Intern Challenge Data Set - Sheet1.csv` — dataset used by the notebook

## How to run

1. Create a Python environment (conda/venv).
2. Install a standard data-science stack (for example: `pandas`, `numpy`, and whatever plotting libs you prefer).
3. Start Jupyter:

```bash
jupyter lab
# or
jupyter notebook
```

4. Open the notebook and run cells top-to-bottom.

## Notes

- This is an **educational / portfolio** repo.
- If you’re using a different Python environment, you may need to adjust imports in the notebook accordingly.
